# CSS
CSS - Cascading Style Sheet.

# How to link css file into HTML

- in HTML we have link tag through which we can link css file.
- <link rel="stylesheet" href="style.css" />


# Change Font or text size
- font-size in pixels.

# change the look of font
- font-family: 'haveltica'

# Weight of the font
- font-weight: 100 - 700

# change the font style
- font-style: italic or bold

# alignment of text to left, center or right
- text-align: left | center | right

# Change the casing of font

- text-transform: lowercase | uppercase | capitalize

# id and class selectors

- id: for unique selector.
- class: for repeating stuffs.

# how to add custom font

- @font-face {
    font-family: myFont;
    src: url('')
}
